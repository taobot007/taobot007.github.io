1. sorted的key 与 cmp

```
def self_defined_sort(x, y):
    if x[1] < y[1]:
        return -1
    if x[1] == y[1]:
        return 0
    else:
        return 1

a = [[1, 2], [2, 1]]

print(sorted(a, cmp = self_defined_sort))
print(sorted(a, key=lambda x: x[1]))
```

2. python 自带binary search

```
from bisect import bisect_left as bl, bisect_right as br
x = [1, 2, 2, 2, 3, 4]
bl(x, 2)   -> 1
br(x, 2)   -> 4
bl(x, 2.5) -> 4
br(x, 2.5) -> 4
br(x, 2, 0, 2) -> 2   # 从0到2-1的range中找
```
